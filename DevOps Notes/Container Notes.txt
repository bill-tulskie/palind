Install ps using apt-get:

# apt-get update && apt-get install procps

Stop and restart the container to stop/restart the Django server

To attach a terminal (iterm or vscode terminal) to the running container: 

> docker exec -it <container ID> sh

e.g. > docker exec -it 6fd2aa1a5fa00adeaba1e0de77adbad98a49a7101988eede210317163c342d3f sh

. . . or start a terminal in the container from docker desktop.

After attach:

# ps

To restart django
# python manage.py runserver
